# Contributing

This repository is currently an independent portfolio/demo project maintained by Muhammad Zaenal Abidin Abdurrahman.

The working style is intentionally incremental:

```text
small scope -> clear commit -> documented progress -> verification
```

Commit messages should follow a concise conventional style, for example:

```text
chore: bootstrap project foundation
feat(web): add user application form
feat(api): add loan application endpoint
docs: update reviewer quickstart
test(api): add loan validation tests
```

Large changes should be split into reviewable checkpoints.
